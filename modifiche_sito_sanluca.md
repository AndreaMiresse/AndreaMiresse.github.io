# Modifiche al sito – San Luca Stadium

Questo documento descrive le modifiche testuali e strutturali da apportare al sito web del San Luca Stadium. Applica ogni modifica con precisione, rispettando i testi forniti. Non modificare aspetti grafici o di stile non esplicitamente indicati qui.

---

## 1. HOME PAGE

Sostituisci il titolo principale con:

```
SAN LUCA STADIUM
```

Sostituisci il sottotitolo con:

```
Il punto di riferimento per il calcio a 7 in Val Bisagno.

Campo di ultima generazione, ampio parcheggio privato, spogliatoi dedicati e partite disponibili sulla piattaforma Replayer.
```

Assicurati che siano presenti due pulsanti principali:

- **Prenota il campo**
- **Contattaci su WhatsApp**

## 8. ELIMINAZIONE FORM DI CONTATTO

**Elimina completamente** il form di contatto (se presente nel sito), inclusi:

- Tutti i campi del form (nome, email, messaggio, ecc.)
- Il pulsante di invio del form
- Qualsiasi logica JavaScript associata all'invio del form
- Il file CSS dedicato al form di contatto (es. `contact-form.css`, `form.css` o equivalente), se esiste come file separato
- Eventuali classi CSS relative al form all'interno di fogli di stile condivisi (rimuovi solo le regole strettamente legate al form, senza alterare il resto dello stile)

L'obiettivo dei contatti è indirizzare l'utente esclusivamente verso telefono e WhatsApp, non raccogliere dati tramite form.
